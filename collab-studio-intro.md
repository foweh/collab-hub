# Collab Studio — 局域网实时协作创作工作室

## 简介

Collab Studio 是一个面向创意团队的**局域网实时协作平台**，支持剧本写作、思维导图、故事创作和分镜脚本等多种创作场景。所有数据在本地网络传输，无需注册外部账号，不依赖云端服务。

---

## 核心功能

### 剧本编辑器
- 按"幕-场-对白"三层结构组织剧本
- 支持角色管理、动作描写、环境描写
- 实时同步：多人同时编辑同一剧本
- 拖拽排序：场次、对白可自由拖拽调整

### 思维导图
- 类 XMind 的 Canvas 渲染引擎
- 拖拽节点、自由缩放、贝塞尔曲线连线
- 支持多种节点形状（矩形/椭圆/菱形）
- 可折叠子树、节点着色、标记系统
- 撤销/重做栈（最多 50 步）
- 导出为 PNG 图片

### 故事编辑器
- 分章节编写故事
- 篇章结构管理
- 实时保存与同步

### 分镜脚本
- 独立的 Vue 3 分镜工具（fenjing/）
- 场景与镜头管理
- 跨设备状态同步

### 局域网多机协作
- UDP 广播自动发现局域网节点（端口 41234）
- Socket.IO 实时数据同步
- 节点断开后 5 分钟重连窗口
- 手动指定加入（--join 模式）
- 设备管理面板，可给每个设备添加备注

### 协作特性
- **在线用户列表**：查看所有协作者
- **操作锁系统**：正在编辑的内容自动锁定，防止冲突
- **操作审计日志**：记录所有创建、修改、删除操作
- **用户间私聊**：管理员批准后可向特定用户发送消息
- **项目传输**：一键将项目发送给其他节点

### 管理功能
- **用户管理**：查看所有用户，修改密码，拉黑/解禁
- **密码重置审批**：用户忘记密码可提交申请，管理员审批
- **消息权限控制**：用户需管理员批准才能发送私聊
- **实时统计**：在线人数、节点数、项目数、日志数

## 快速开始

```bash
# 安装依赖
cd collab-studio
npm install

# 启动服务（默认端口 3000）
node server.js

# 或指定端口 + 加入已有的工作室
node server.js --port 3001 --join 192.168.1.100:3000
```

打开浏览器访问 `http://localhost:3000`，输入名字即可进入。

## 安全特性

### 密码安全
- 密码使用 **bcryptjs** 加盐哈希存储（非明文）
- 首次启动自动生成管理员密码（支持 `ADMIN_PASSWORD` 环境变量）
- 首次登录**可免密注册**；一旦设置密码，后续必须验证
- 所有敏感文件（data/ 目录）权限设为 600/700，且被 .gitignore 排除

### 访问控制
- **速率限制**：登录最多 20 次/分钟/IP，密码重置最多 3 次/5 分钟/IP
- **指纹拉黑**：管理员可按浏览器指纹封禁设备
- **管理员审批**：密码重置、私聊权限均需管理员批准

### 数据保护
- 所有 JSON 数据文件（用户、项目、日志）存储在 `data/` 目录
- 该目录通过 Express static 隔离，无法通过 HTTP 直接访问
- 操作日志保留最近 500 条，自动滚动清理

## 技术栈

| 层 | 技术 |
|--|--|
| 后端 | Node.js + Express + Socket.IO |
| 前端 | 原生 JavaScript (ES Modules) |
| Canvas 渲染 | 思维导图自研引擎 |
| 网络发现 | UDP dgram 广播 |
| 实时同步 | Socket.IO 事件驱动 |
| 数据持久化 | JSON 文件存储 |
| 密码加密 | bcryptjs |

## 开发背景

本项目最初由一个创作者在业余时间独立开发，经历了多次功能叠加后积累了较多的技术债务（"屎山代码"）。2024 年末进行了安全重构，主要修复了：

- 移除源码中硬编码的管理员密码，改用 bcryptjs
- 解除 sessionStorage 密码明文存储
- 添加频率限制防止暴力破解
- 代码模块化拆分，消除大量重复定义
